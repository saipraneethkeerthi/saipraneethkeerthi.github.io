Smart Vehicle Monitoring System

In this project, smart vehicle is implemented to monitor the system using GSM. If any crash occur, then SMS will be sent to the owner or to saved contacts. Most of the road accidents are happened due to alcohol drinking and not obeying the traffic rules. If toxic gas present in any AC cars, then toxic gas sensor will sense the gas and the indication is displayed by using light then the window door will be automatically opened.

Automatic Street Lights Using IR Sensor

In this project, consumption of electricity is reduced using IR sensor. During day time many street lights been glowing until it is turned off manually. So in order to reduce that Light Dependant Resistor (LDR) is used to sense the light and that sends the signal to sensor to turn off the light. The main advantage of this project is that the light is turned on/off automatically with the help of IR sensor during the presence/absence of people roaming on road.
